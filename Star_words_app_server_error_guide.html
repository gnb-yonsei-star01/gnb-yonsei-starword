GnB연세스타 권곡캠퍼스 · 학부모님 안내용

안녕하세요, 학부모님 😊
우리 아이가 단어 학습 앱을 쓰다가 접속이 안 되거나 소리가 안 날 때, 아래 순서대로 한 번씩만 따라 해 주시면 대부분 바로 해결됩니다. 어렵지 않으니 천천히 확인해 주세요!

⚡ 급할 때 먼저 이것부터!
1.앱을 완전히 닫았다가 다시 열기
2.와이파이 ↔ 데이터(LTE) 바꿔서 접속
3.그래도 안 되면 → 네이버 웨일 브라우저 설치 후 접속
4.소리 안 나면 → 폰 볼륨 키우고 화면 한 번 터치
📡
접속이 안 될 때
💡 크롬에서 안 들어가지는 경우가 가장 많아요. 아래 순서대로 해보세요.
1
크롬 인터넷 기록(캐시) 지우기
오래된 정보가 남아 충돌하는 경우가 많습니다. 한 번 비워주면 깨끗해져요.
크롬 ⋮ 메뉴▸ 설정▸ 개인정보 보호▸ 인터넷 사용 기록 삭제
2
'데이터 절약 모드'가 켜져 있으면 끄기
이 기능이 켜져 있으면 일부 기능이 막힐 수 있어요.
크롬 ⋮ 메뉴▸ 설정▸ 데이터 절약 모드 끄기
3
와이파이를 끄고 데이터(LTE/5G)로 접속
집·학교 와이파이가 특정 사이트를 막는 경우가 있어요. 데이터로 바꿔보면 구분됩니다.
✅
그래도 안 되면 '네이버 웨일' 브라우저를 설치해 주세요!
플레이스토어(안드로이드) 또는 앱스토어(아이폰)에서 '네이버 웨일'을 검색해 설치한 뒤, 학원에서 보내드린 주소를 입력하면 잘 열리는 경우가 많습니다. 크롬이 막혀도 웨일 브라우저는 되는 경우가 흔해요.
📅
폰 날짜·시간이 맞는지도 확인해 주세요.
시간이 틀어져 있으면 보안 문제로 접속이 막힙니다. 설정 → 날짜 및 시간 → 자동 설정 켜기로 해두시면 안전해요.
🔊
소리(발음)가 안 날 때
💡 우리 앱은 폰에 들어있는 음성 기능으로 영어를 읽어줍니다. 접속과는 별개의 문제예요.
1
폰이 무음/진동 모드인지 확인
옆면 버튼이나 알림창에서 무음을 풀고, 미디어 볼륨을 올려주세요.
2
앱에 들어가면 화면을 한 번 터치하기
브라우저는 처음에 소리를 막아두는데, 화면을 한 번 누르면 풀립니다. 시작 후 아무 곳이나 톡 눌러주세요.
3
영어 음성이 폰에 깔려 있는지 확인 (안드로이드)
음성 데이터가 없으면 소리가 안 나요. 아래 경로에서 '영어'가 있는지 확인해 주세요.
설정▸ 소리/접근성▸ 화면 읽기(TTS)▸ 언어에 영어 추가
🎧
다른 앱(유튜브 등)에서는 소리가 나나요?
• 다른 데서도 안 나면 → 폰 볼륨·무음 문제예요.
• 다른 데선 잘 나는데 우리 앱만 안 나면 → 위 3번(영어 음성)을 확인해 주세요.
🛡️
자녀보호 앱을 쓰고 계신 경우
💡 자녀보호·시간관리 앱이 학습 사이트를 '모르는 사이트'로 보고 막는 경우가 있어요.
★
아래 3개 주소를 '허용 목록'에 추가해 주세요
보호 기능을 통째로 끄지 않고, 학습 사이트만 통과시킬 수 있어요. 자녀보호 앱의 '허용 사이트' 또는 '화이트리스트' 메뉴에서 추가하시면 됩니다.
script.google.com
script.googleusercontent.com
학원에서 보내드린 앱 주소
💛
자녀보호 앱 종류(구글 패밀리링크, 엑스키퍼, 모비커브 등)에 따라 메뉴 이름이 조금씩 달라요. 어려우시면 어떤 앱을 쓰시는지 학원에 알려주시면 방법을 자세히 안내해 드리겠습니다.
